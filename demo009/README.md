## 前言
`awaken`项目致力于基于Java基础上,集成各种技术框架,用于理解框架集成原理。

## 项目文档
文档地址：[https://github.com/Awaion/tools](https://github.com/Awaion/tools)

## 当前项目介绍
- `Struts`, `Spring`, `MyBatis`框架集成
- 需要依赖`MySQL`, `Tomcat`
- 这是一个古老的项目
