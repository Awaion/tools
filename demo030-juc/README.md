## 简介

Doug Lea 纽约州立大学Oswego分校教师,java.util.concurrent 包作者

为什么会出现多线程
- 硬件方面:摩尔定律失效,主频不能翻倍,CPU进入多核时代
- 软件方面:互联网流量爆炸,高并发系统出现

多线程问题
- 线程安全性问题
- 线程锁问题
- 线程性能问题
